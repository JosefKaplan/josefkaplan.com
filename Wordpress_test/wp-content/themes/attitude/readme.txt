== Description ==

Attitude is a simple, clean and responsive WordPress theme which is compatible to popular plugins like WooCommerce, bbPress and many more.

== Installation ==

= Manual installation: =

1. Upload the `attitude` folder to the `/wp-content/themes/` directory

= Activiation and Use =

1. Activate the Theme through the 'Themes' menu in WordPress
2. See Appearance -> Customize to change theme specific options

== Frequently Asked Questions ==

= How to use featured Post Slider? =

* Create Post, Add featured image to the Post.
* Add all the Post ID that you want to use in the featured slider. (You can now see the Posts' respective ID in the All Posts' table in last column.)
* Featured Slider will show featured images, Title and excerpt of the respected added post's IDs.
* You can now add Pages ID too. (You can now see the Pages' respective ID in the All Pages' table in last column.)

= Post ID shown in the All Pages/Posts Table=

* This theme shows the post id of the pages/posts in the all pages/posts table in the dashboard.
* This is added to help the users to use those ids in the slider option as mentioned above.
