== Description ==
 ScrollMe - A horizontal scrolling WordPress Themes lets you navigate your website horizontally. The theme has beautifully crafted clean and elegant design and is fully responsive that show beautifully on all the devices. ScrollMe is a multipurpose theme and is perfect for business, web agency, personal blog, portfolio , photography, magazine, parallax one page and freelancer. ScrollMe has powerful features and provides a easier way to configure the front page with live preview from a customizer panel. The theme is Polylang compatible, Woo Commerce compatible, Translation Ready, SEO Friendly, bbPress and works with all other major plugins as well. So look no further and start creating your beautiful website using ScrollMe. For demo http://demo.accesspressthemes.com/scroll-me

Install Steps:
--------------

1. Activate the theme
2. Go to the Theme Option page
3. Setup theme options

Configuring Featured Home Page
------------------------------
1. Create Home Page
    - Go to Dahboard > Pages > Add New.
    - Set the title for the Home Page.
    - Select the 'Home' page template.
    
    - Now Go to Dashboard > Settings > Reading.
    - Set the 'Front Page' to diplay a Static page, then select home page you currently created from 'home page' dropdown list.
    - 
    
2. Configuring Featured Scroll Home Page
    A. Configure Home Page.
        i) Configuring Slider Section.
            # Add Category for Slider
            - Go to Dashboard > Posts > Categories.
            - Enter the Name for the slider Category (e.g. 'Sliders')
            
            # Add Slider Posts
            - Go to Dashboard > Posts > Add New.
            - Enter the title for the slider post (displayed as slider caption title.
            - Enter the contents for the slider post (displayed as slider caption description.
            
            # Configure Slider.
            - Now go to Dahsboard > Appearance > Customizer > Scroll Page Settings > Scroll Section - Home.
            - Choose the Slider Category from 'Choose category for slider' dropdown list.
            
        ii) Configuring Section (1-6)
            - You may configure to display any pages whether with static contents for ones created using custom page templates in Section (1-6).
            - For this Go to Dashboard > Pages > Add New.
            - Create a Page (either with custom template assigned to it or not)
            - Creating a page with custom page templates provie you with the unique set of features. For e.g. creating page with 'Contact' will provide you the ability to display the wiget contents of the 'Google Map' widget in the section.  
            - You're also provided with the feature to set the 'ID for Navigation' for each section you configure.
            - These id's will be used to navigate to a respective section by clicking a menu item.
            - For this you'll need to assign a custom link to the primary menu. (e.g. http://localhost/themes/scrollme/#home/spage)
            - Then you'll need to assign 'spage' as the section's id for the respective section.
            - Doing this you'll have the ability to navigate to the respective section by clicking it's respective menu item.
